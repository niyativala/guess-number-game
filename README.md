# 🎯 Guess Number Game

A fun JavaScript game where users guess a random number within a time limit.

---

## 🚀 Features
- Random number generation
- Timer-based challenge ⏱️
- Hint system (too high / too low)
- Interactive alerts

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📂 Project Files

* `index.html` → Game structure and UI
* `style.css` → Styling and layout
* `script.js` → Game logic and functionality

---

## 🎯 How Game Works

1. Enter a number in the input field
2. Click the guess button
3. Computer checks your guess
4. Result is displayed:

   * 📈 Too High
   * 📉 Too Low
   * 🎉 Correct Guess
5. Keep trying until you win

---

## ▶️ How to Run Project

1. Download or clone this repository
2. Open `index.html` in your browser
3. Start playing 🎮

---

## 📌 How to Play
1. Enter max number
2. Guess the number
3. Win before time runs out!
