# 🎯 Guess Number Game

A fun JavaScript game where users guess a random number within a time limit.

---

## 🚀 Features
- Random number generation
- Timer-based challenge ⏱️
- Hint system (too high / too low)
- Interactive alerts

---

## 🛠️ Tech Used
- HTML
- JavaScript

---

▶️ How to Run Project
Download or clone repository
Open index.html in browser

---

## 📌 How to Play
1. Enter max number
2. Guess the number
3. Win before time runs out!
